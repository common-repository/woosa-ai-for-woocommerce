Generate, edit, and publish AI product information all in one platform with Woosa AI.

== Description ==

Woosa AI for WooCommerce generates product information, such as:

* Short description
* Description
* Focus keyword (SEO)
* Meta description (SEO)

The plugin is compatible with Yoast SEO.

The functionalities of this plugin are easy:

* Push products in bulk to the Woosa AI platform so that content can be generated.
* Publish the generated content automatically from Woosa AI to WooCommerce.

Regular AI tools generate, Woosa AI automates.



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/integration-woosa-ai-for-woocommerce` directory, or install the plugin through the WordPress plugins section directly.
1. Activate the plugin through the 'Plugins' section in WordPress.
1. Navigate to 'Woosa AI' in the main dashboard menu to configure the plugin.
